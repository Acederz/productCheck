# 管理员怎么用（傻瓜版）

你是管理员。别人填分类，你负责「发活」和「验收」。

打开网页 → 输入账号密码 → 登录。

---

## 平时你要做的事（按顺序）

### ① 给同事开账号（新人入职时做）

1. 点左边 **用户管理**
2. 点 **新建**
3. 写上用户名、密码，角色选 **操作员**
4. 保存，把账号密码告诉同事

同事走了：在用户管理里把账号 **停用**。  
同事忘密码：你帮他 **重置密码**。

---

### ② 上传「分类规则」表（一般很少做）

规则表决定：操作员下拉里能选什么。

1. 点左边 **分类规则**
2. 点导入，选规则 Excel
3. 等提示成功

什么时候再做一次？领导说规则变了、换了新表的时候。

---

### ③ 上传「商品」表（经常做）

1. 点左边 **数据导入**
2. 选商品 Excel，上传
3. 看成功了多少、失败了多少
4. 失败多：下载错误说明，改完再传

传成功后，这些商品会出现在 **任务管理** 里。

---

### ④ 把活分给操作员（经常做）

1. 点左边 **任务管理**
2. 上面筛一下：比如「未分发」
3. 勾选要分的商品
4. 点 **分配**，选哪位同事
5. 确定

分完后，同事在自己的「我的任务」里就能看到。

---

### ⑤ 检查同事填好的内容（经常做）

1. 点左边 **审核中心**
2. 一条一条看（可看图片、名称、填的分类）
3. 对的点 **通过**
4. 不对点 **驳回**，写清楚哪里错了，方便同事改

驳回后同事会改完再交过来，你再审一遍。

---

### ⑥ 看已经通过的数据 / 导出表格

- **正式数据**：已经通过的，在这里查
- **数据导出**：需要交 Excel 时，在这里下载

---

## 每天建议怎么干

| 什么时候 | 做什么 |
|----------|--------|
| 早上 | 上传新商品表 → 分给同事 |
| 白天 | 有空就进审核中心看看 |
| 下班前 | 把待审的尽量审完；要交数就导出 |

---

## 出问题怎么办

| 情况 | 你怎么处理 |
|------|------------|
| 同事说看不到任务 | 看是不是还没分给他；账号是不是停用了 |
| 同事说下拉框是空的 | 看规则表有没有上传成功 |
| 上传失败很多 | 下载失败说明，对照改 Excel 再传 |
| 自己忘了密码 | 找会技术的同事帮忙，或用别的管理员账号重置 |

---

## 记住这一句话

**上传商品 → 分给同事 → 审核通过或驳回 → 需要时导出。**
